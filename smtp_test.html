üìù Before Testing:

  1. Open contact.php in your text editor
  2. Find the $smtp_config array (around line 47)
  3. Update these values with your email provider's SMTP settings:
    • host - Your SMTP server (e.g., mail.lsuczm.com)
    • username - Your email address (admissions@lsuczm.com)
    • password - Your email password
    • port - Usually 587 for TLS or 465 for SSL
  4. Save the file and test below!

üîç What happens when you click "Send Test Email":

  1. First attempt: PHP tries to send via basic mail() function
  2. If that fails: PHP tries SMTP with your configured settings
  3. If both fail: Error is logged to application_logs.txt
